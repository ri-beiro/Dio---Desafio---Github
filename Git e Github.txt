Por baixo dos panos do GIT :

SHA1 = significa Secure Hash Algorithm (algoritmo de HASH Seguro) 
é um conjunto de funções hash criptograficas projetadas pela NSA do EUA.
um conjunto de 40 caracteres criptografados. A cada virgula que voce muda no arquivo que contem 
o SHA1, a encriptação é mudada completamente!

HASH = identificação da string, bloob, tree, commit.


BLOB : é um objeto que guarda o conteudo dos objetos, com tamanhos e conteudos.
guarda o Sha do arquivo

TREE : Armazenam os 'BLOBs', apontando para um deles ou blobs ou outras Trees, ela guarda o nome do arquivo

COMMIT : é o objeto que junta tos, que da sentido a tudo, aponta para as trees, parente, autor, mensagem
